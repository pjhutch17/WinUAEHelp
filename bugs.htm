Known Bugs

WinUAE has some bugs which are known. Please check this list before reporting any bugs:

  • Some CD32 games stop loading and freeze (bug in CDROM controller emulation, hard to fix because there is no CD32 hardware specs..)

  • Sound and JIT-sound timing problems (some systems only)

  • CDTV CDROM controller and drive is not emulated yet (missing feature).


    If you find any bugs, then include the following information with the bug report:

    1. Amiga OS versions (kickstart and workbench)
    2. Extensions running (MagicWB, NewIcons, DirOPUS, ToolManager, etc.)
    3. Settings of Amiga that caused the problem
    4. HostOS (Win95, Win95-OSR2, Win2K, etc.)
    5. DirectX version
    6. PC Graphics card and its driver's version
    7. Sound card and its driver's version
