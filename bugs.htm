Known Bugs

WinUAE has some bugs which are known. Please check this list before reporting any bugs:

  • some CD32 games stop loading and freeze (bug in CDROM controller emulation, hard to fix because there is no CD32 hardware specs..)

  • sound and JIT-sound timing problems (some systems only)

  • CDTV CDROM controller and drive is not emulated yet (missing feature).
