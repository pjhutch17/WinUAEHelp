Amiga Programs

Along with the WinUAE emulator program, some programs are supplied in the 'Amiga Programs' folder in the C:\Program Files\WinUAE folder. Here is an explaination of what these files are for and where to copy them to:

MouseHack

Copy this program file your C directory on your Workbench disk or hard disk. This program will hack your mouse connected to your PC to work with WinUAE. In most cases it may not be needed. You can auto run it by adding a line to the s:user-startup file on your Workbench disk.

Picasso96fix

Copy this program file your C directory on your Workbench disk or hard disk. This program will fix Picasso 96 RTG screens greater than 1024 pixels when using your PC's graphics card with the uaegfx driver in WinUAE. Load it before LoadWB in the startup-sequence. The Picasso96fix.bb file is the source code to this program written in Blitz Basic.

rtg.library

Re-targetable Graphics Library. Copy this library file your libs:picasso96. This library will avoid mousetrails and redraw problems in picasso96 when using JIT functions.

timehack

This setst the correct time every second,useful when you hybernate your PC when winuae is running.

transdisk

Copy this program file your C directory on your Workbench disk or hard disk. This can Transfer Disk images to or from floppy disk to an image or vice versa in ADF (Amiga Disk Format). Type transdisk ? for more options. See trandisk page for more details on this program.

transrom

Copy this program file your C directory on your Workbench disk or hard disk. This can Transfer the Kickstart rom image from a real Amiga to an image file to use with your Amiga Emulator. It will produce either a 256K or 512K sized file. See transrom page for more details on this program.

UAE

Copy this file to the AHI driver drawer in Devs on your Workbench disk or hard disk. It is used to emulate the Amiga's sound system using AHI retargetable sound driver system.

UAE.audio

Copy this file to the Devs:AHI driver drawer in Devs on your Workbench disk or hard disk. It is used to emulate the Amiga's sound system using AHI retargetable sound driver system. It will support 44.1khz 16Bit and 32khz.

UAE_German

This is a German keymap file for access to @\~{[]} with ALT-GR like windows. Copy it to devs/keymaps and select in Input prefs.

uae_rcli

Remote Cli. Usage: uae_rcli [?|-h] [-debug] [-nofifo] [<delay>]

uae-control

Copy this program file to your Utilities folder or some other folder on your Workbench disk or hard disk. It will provide you a user interface within the emulator to control the floppy drives, reset, quit and other features without having to press F12 to bring up the WinUAE interface.

uaectrl

Copy this program file your C directory on your Workbench disk or hard disk. This is the Shell equivalent of the uae-control program which you can change floppies, sound, joystick, reset or quit the emulator from a simple text menu.

winuaeclip

This program allows you to have access to Windows Clipboard so you can copy/paste files to/from WinUAE.

winuaenforcer

This program allows Enforcer support, to detect bad memory access when developing programs or debugging.
For 100% safety, in JIT use Indirect Mode and set Speed to 2.

winxpprinthelper

Not sure what this does yet. Need printer switched on before running this.

uae_german

This is a German keymap to reach @\~{[]} with ALT-GR like windows. Copy it in devs/keymaps and choose it in the keymap-prefs

p96refresh (winuae only)

Start this program to get higher mousepointer redraw rates than 50 hz

with P96.Actions are seen immediatly after call.You can start this in your

user-startup for automatic on boot

 

Example:

p96refresh 75

set redraw rate to 75 hz.Best is if you set rate

to your monitor refresh rate or monitor refreshrate*2

Dont set it higher or you get speedloss > 2%
