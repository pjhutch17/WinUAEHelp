Compatibility

 

According to the testers and the users of WinUAE, it proved itself worthy. Compatibility for both UAE and it's Windows port should be really very close to 100%.

If you have problems to get a certain program working, look at the Links. There you will find the most comprehensive emulation sites on the net.

The number of programs that don't run has been reported to be close to zero. If you find a piece of software to require special treatment, please let the developers (Bernd Schmidt for UAE, the current WinUAE maintainer is Toni Wilen - see Contacts) know.
