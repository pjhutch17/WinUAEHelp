Command-Line Parameters



UAE's command line options are quite hidden in WinUAE, since it features a mighty GUI to visualize the options and settings that can be set in UAE from the command line.

There are two parameters left:
  • -config=configfile.uae

  • -f configfile.uae

which will both do the same thing: they will cause WinUAE to load the given Configfile during WinUAE is starting.
