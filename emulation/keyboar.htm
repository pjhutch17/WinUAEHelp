Keyboard Emulation

Most of your keyboard retains its regular functionality under UAE, but there are a few notable exceptions:

  • If you don't have a Win95 keyboard, you will have to use Ins/Home as a replacement for Left Amiga/Right Amiga. Also, if you are using WinNT with a Windows keyboard, you will STILL have to use Insert/Home as your "Amiga" keys.
  • HELP has been remapped to Page-Down.
  • F12 brings up the GUI.
  • PRINTSCREEN copies WinUAE's current screen to clipboard
  • END + PrintScreen: save screenshot file (was F11 in older versions)
  • END + F5: restore state
  • SHIFT + END + F5: save state
  • PAGE UP: Action Replay "freeze"-button
  • SCROLL LOCK toggles screen refresh, speeding up the emulation.
  • PAUSE toggles sound, speeding up the emulation (NOTE: you can't enable sound this way if you haven't configured UAE to run with sound from the beginning).
  • END+F1, F2, F3 or F4 allows you to change disks in one of the four Amiga disk drives.
  • SHIFT+END+F1...F4 ejects the disk.
