The GUI



WinUAE includes a fairly powerful (and complicated) GUI (= Graphical User Interface) to manage the various settings and abilities of your virtual "Amiga". Click on the different tabs in the GUI below to see the help for that particular page of the GUI, or use the list below the picture.


The GUI contains everything you will need to configure WinUAE:

  • Paths
  • Quickstart
  • Configurations
  • Hardware
  • CPU
  • Chipset
  • ROM
  • RAM
  • Disk Drives
  • Hard Drives
  • Host
  • Display
  • Sound
  • Game and I/O Ports
  • Input
  • Output
  • Filter
  • Disk Swapper
  • Misc.
  • Priority
