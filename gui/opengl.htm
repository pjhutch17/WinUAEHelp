Direct3D/OpenGL Filters


The OpenGL screen now provides 3D display support from the Windows OpenGL drivers and can be configured via this screen. Here, you can change the size and position of the Amiga screen, useful for games with limited resolutions.

Tick the Enable box to enable Filter support.

You can specify whether to use Null Filter, Direct3D, OpenGL, Scale2X, SuperEagle, Super2XSal or 2xSal graphics support with Nofilter or Bilinear graphics in either 16 or 32 bit modes.

The Horizontal and Vertical Sizes can be specified from -50 to 50.

The Horizontal and Vertical Positions can be specified from -50 to 50.

The Scanlines can be specified from 1:1 to 3:3 and from 0 to 100.

Scanline Brightness can be specified from 0 to 100.

Best results can be set by using low resolution and disable line-doubling.
