Expansions

This will change settings for expansion cards such as RTG, Network, and SCSI cards.

 

RTG Graphics Card Settings

  • RTG Graphics Card to select a Zorro II, Zorro III
    • , Picasso II, Piccalo, EGS-28/24 or Picasso IV card.
  • RTG Memory is for graphics card use such as the Picasso96 Retargetable Graphics system so that you can use your PC's graphics card rather than the custom chips for extra speed. Upto 512 MB
  • Match host and RTG color depth if possible Color space conversion enabled.
  • Scale if smaller than display size setting Rescale display if current window is smaller than the display size setting given
  • Always scale in windowed mode Rescale display instead if using windowed mode.
  • Always center Will make sure RTG display is centered in middle of monitor.
  • Hardware vertical blank interrupt emulation
  • Hardware sprite emulation
  • Montor selection menu can be used if you have more than one monitor and need to select another.
  • Refresh Rate This can be Disabled, Chipset, Real or 50, 60, 70 or 75 Hz.
  • Buffer Mode This can be Double or Tripe Buffering for graphics.
  • Aspect Ratio This can be Disabled, Automatic, 4:3, 5:4, 15:9, 16:10,
    • 27:16, 128:75, 16:9, 256:135, 21:9 or 16:3.

 

Miscellaneous Expansions

  • Catweasel Z2 emulation Emulates the Catweasel floppy disk controller using Zorro 2 slot (moved from Misc in v2.0)
  • uaescsi.device Adds support for IDE/SCSI CDROM drives on Amiga side. Use this device name in your CD0 DosDriver file in Devs\DosDrivers, and add the appropiate unit number (use Create Log file to see which unit number your CD drive uses).
    First CDROM is always uaescsi.device unit zero, second is zero etc.. and then other non-CDROM SCSI devices (moved from Misc in v2.0).
  • A590/A2091 WD33C93 SCSI Add support for A500's A590 external hard drive or A2000's A2091 SCSI devices.
  • A4091 NCR53C710 SCSI Add support for A4000's SCSI devices.

 

Network

  • bsdsocket.library Allows your Amiga to use your PC's network connection, whether it be through Dial-Up Networking (DUN), or Ethernet connectivity. In other words, if your PC is connected to your Internet Service Provider (ISP) through a modem, or connected to a LAN through an Ethernet connection, your Amiga will be able to make use of that connection (moved from Misc in v2.0).
  • uaenet.device Enable SANA2 compatible net device. Requires WinPCap (Windows Packet Capture Library) (moved from Misc in v2.0).
  • A2065 Z2 Emulates the A2065 Zorro 2 card. Option below is 'Cable Disconnected'.
