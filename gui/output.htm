Output


The Output tab allows you record video and audio (from selected Codec's) to an AVI file.


Output Name of output file to save to (default is C:\Output.avi).

Video Type of Video codec used to create/compress the video file.

Audio Type of Audio codec used to create/compress the audio data.

Disable frame rate limit while recording - Removes any limits when recording certain types of screen

Playback Rate - Set to PAL or NTSC modes (50/60 fps)

Save screenshot Allows you to take a snapshop of whats on the screen while emulation is running. Ideal for taking snaps
of your latest high scores or your desktop easily. Use can use the F11 key as well.

Pro Wizard Rips music scores.
