Input

You can configure your Input device such as Joysticks to match your requirements for games or other software:

You can specify Compatibility mode with standard Windows setup or specify upto 4 configurable settings yourself. For each Input
Source you can configure each Input Target using options from the pull down menu. For example you can specify whether its a
Joystick, Mouse, Keyboard, Lightpen, Lightgun or Parallel Joystick or whether its a Horizontal, Vertical, Button or direction.

The Autofire option can be specified for any of the buttons. There is a tick box at the end which you can disable or enable Configuration if needed.

The Joystick dead zone can be specified for a percentage of the Joystick's movement to prevent unwanted movement.

The Autofire rate (frames) can be specified for how often firing occurs over a number of frames.

The Digital joy-mouse speed can be specified for how fast response is for digital input devices (e.g. USB).

The Analog joy-mouse speed can be specified for how fast response is for analogue input devices (e.g. MIDI port).

The Mouse Speed can be specified for how fast response is for the standard mouse (default is 100).

The Copy From option allows you to copy a configuration from another configuration into the current configuration to save time.

The Swap 1<>2 button allows you to swap between joystick/mouse ports.
