Priority

When Active

When the program is active and running you can set priority to different priorities to improve performance of Winuae than other programs.
Settings include low, Below Normal, Normal, Above Normal and Highest.

When InActive

When the program is active and running you can set priority to different priorities to improve performance of other programs when Winuae is not in use.
Settings include low, Below Normal, Normal, Above Normal and Highest.

Disable sound output turns off audio output when minimized.

Pause emulation will completely stop emulation when Winuae is minimised.

When Minimized:

Run at lower priority makes the emulation use less CPU resources when Winuae is minimized. You can specify the level of priority to use:
Highest, Above Normal, Normal, Below Normal, and Low.

Disable sound output turns off audio output when minimized.

Pause emulation will completely stop emulation when Winuae is minimised.
