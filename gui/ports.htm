Game and I/O Port Settings

These settings let you control how joystick support is provided in WinUAE, as well as choosing serial and parallel port emulations.



Serial:

You can adjust the serial port (COM1....) to be used as serial port within UAE. Only the ports which are available on your PC are selectable.
RTS/CTS handshaking can be enabled if its required for modems etc. If Direct is enabled then it will fixes PC to PC Lotus 2 serial link
problems when using serial port.

MIDI In/Out

Here you can select the MIDI - device(s) to be used. Only available when the Serial- emulation is enabled.

Printer

This lets you specify which PC installed printer you want to use for printing for the "Amiga".

Amiga Mouse Port 0 / 1

With these 2 options you can select the configuration of the "Amiga" - ports. Port 0 is usually used by the Amiga - mouse. It is easy to exchange the settings with the swap button.
