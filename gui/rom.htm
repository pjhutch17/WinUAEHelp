ROM

These settings control which Kickstart ROM-image WinUAE will use.

ROM Settings

  • Boot ROM File specifies which Kickstart ROM file to use. This contains essential parts of the Amiga Operating system (usually 512kB or 1MB big). The ROM file can be compressed to .zip or .roz files if needed. The ROM file can be grabbed using the transrom utility from a real Amiga or copied from the Amiga Forever package. It can auto detect which roms you have (Encrypted ones only).
  • Extended ROM File will provide CDTV or CD32 support (supplied with Amiga Forever).
  • MapROM Emulation will provide support for BlitzKick type tools.
  • ShapeShifter support will enable the ShapeShifter 'PrepareEmul' support, so you don't have to KickShift your ROM-images. This also allows Amiga Forever users (encrypted ROM-images can't be KickShift'ed) to use ShapeShifter.
    NOTE: ShapeShifter is a Mac - emulator for the Amiga. If you didn't know that, don't use this option!

 

Misc Settings

  • Cartridge ROM File will allow to add Action Replay ROM files. From the pull down menu you can select Freezer: HRTMon 2.30 or the Freezer: Action Cartridge Super IV Professional or use the ... to select a third party rom file.
  • Flash RAM File will allow you to save to NVRAM (Non-Volatile RAM) like the CD32

 

HRTMON Settings

  • Enable HRTMon will enable the HRTMon Monitor or Debugger program if installed (for Winuae 1.4.2 or earlier, for 1.4.3 it has been moved to the Cartridge ROM File).
